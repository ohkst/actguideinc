# POEAct - Path of Exile Act Guide

Path of Exile 1 엑트 가이드 iOS 앱 배포용 웹페이지

## 앱 정보

- **앱 이름**: POEAct
- **버전**: 1.0.0
- **플랫폼**: iOS 17.0+

## 주요 기능

- Path of Exile 1 엑트 1-10 가이드
- 스피드런 기준 체크리스트
- 진행률 추적 및 통계
- 팁과 부가설명 제공
- 웨이포인트, 포탈, 전직퀘스트 강조

## 설치 방법

1. 웹페이지에서 "앱 설치하기" 버튼을 탭합니다
2. 설치 확인 대화상자에서 "설치"를 선택합니다
3. 설치가 완료되면 홈 화면에서 앱을 실행할 수 있습니다
4. "신뢰할 수 없는 개발자" 오류가 발생하면 설정 > 일반 > VPN 및 기기 관리에서 개발자를 신뢰하세요

## 배포 파일

- `index.html`: 메인 웹페이지
- `manifest.plist`: 앱 설치 매니페스트
- `POEAct.ipa`: 앱 설치 파일 (별도 업로드 필요)
- `icon-57.png`: 앱 아이콘 57x57 (별도 업로드 필요)
- `icon-512.png`: 앱 아이콘 512x512 (별도 업로드 필요)

## GitHub Pages 설정

1. GitHub 리포지토리 설정 > Pages로 이동
2. Source를 "Deploy from a branch" 선택
3. Branch를 "main" 또는 "gh-pages" 선택
4. 저장 후 웹페이지가 활성화됩니다

## 참고

- IPA 파일은 별도로 업로드해야 합니다
- HTTPS를 사용해야 iOS에서 설치가 가능합니다
- Enterprise 인증서가 필요합니다
